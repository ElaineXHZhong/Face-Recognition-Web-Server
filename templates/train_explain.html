Important: Training set arrangement instruction


1. If you choose image to arrange training set

The directory structure should look like the format below:

|-KOL/
|----Anna/
|--------Anna_0001.png/
|--------Anna_0002.png/
|--------Anna_0003.png/
|----Bob/
|--------Bob_0001.png/
|--------Bob_0002.png/
|--------Bob_0003.png/

Finally, zip the root folder KOL to get KOL.zip and select "Upload image zip file for training" to upload the KOL.zip file.

Notice: All image files should be suffixed with ['png', 'jpg', 'jpeg'] only! Other suffix names are not allowed!


2. If you choose video to arrange training set

The directory structure should look like the format below:

|-KOL/
|----Anna/
|--------Anna_0001.mp4/
|--------Anna_0002.mp4/
|--------Anna_0003.mp4/
|----Bob/
|--------Bob_0001.mp4/
|--------Bob_0002.mp4/
|--------Bob_0003.mp4/

Finally, zip the root folder KOL to get KOL.zip and select "Upload video zip file for training" to upload the KOL.zip file.

Notice: All video files should be suffixed with ['flv', 'avi', 'mp4', 'mov', 'wmv'] only! Other suffix names are not allowed!
